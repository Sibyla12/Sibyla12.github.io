Proyectos Destacados

Factor Analysis

Análisis Multivariado

Implementación de Análisis Factorial para identificar variables latentes y reducir la dimensionalidad de conjuntos de datos complejos. Este proyecto explora las relaciones subyacentes entre múltiples variables observadas, permitiendo una comprensión más profunda de la estructura de los datos.

El análisis incluye la extracción de factores, rotación varimax y evaluación de la adecuación del modelo mediante pruebas estadísticas como KMO y Bartlett.

Python scikit-learn pandas matplotlib

Discriminant Analysis

Clasificación Supervisada

Desarrollo de modelos de Análisis Discriminante Lineal (LDA) y Cuadrático (QDA) para clasificación de datos en categorías predefinidas. Este proyecto demuestra la capacidad de separar grupos mediante la creación de funciones discriminantes óptimas.

Incluye validación cruzada, matrices de confusión y evaluación de métricas de rendimiento para determinar la precisión y efectividad del modelo en diferentes escenarios de clasificación.

Python scikit-learn NumPy seaborn

Clustering Analysis

Aprendizaje No Supervisado

Aplicación de técnicas de clustering como K-Means, clustering jerárquico y DBSCAN para descubrir patrones naturales en los datos sin etiquetas previas. Este proyecto identifica agrupaciones significativas y segmentaciones en conjuntos de datos multivariados.

El análisis incluye la determinación del número óptimo de clusters mediante el método del codo y el coeficiente de silueta, junto con visualizaciones interpretativas de los resultados.

Python scikit-learn pandas plotly
